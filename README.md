# 🌦️ KhetMausam – Weather Alert System for Farmers

A modern **Weather Alert System** designed to help farmers make informed agricultural decisions through **real-time weather updates, crop-specific advisories, weather forecasts, and emergency alerts**.

The project provides an intuitive and responsive dashboard that displays weather conditions, severe weather warnings, farming recommendations, and simulated SMS/WhatsApp alert subscriptions.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/weather-alert-system.git
```

Open project

```bash
cd weather-alert-system
```

Run a local server

Using Python

```bash
python -m http.server 8000
```

Visit

```
http://localhost:8000/weather_alert_system%2045.html
```
